## 📌 Descripción

Este repositorio contiene una colección de **ejercicios y retos de JavaScript** realizados como parte del programa de formación en la academia **Conquer Blocks**.

El objetivo es reforzar los fundamentos de programación en JavaScript mediante práctica constante, resolución de problemas y lógica aplicada.

---

## 🎯 Objetivo del repositorio

- Fortalecer la lógica de programación
- Practicar JavaScript desde básico hasta intermedio
- Mejorar resolución de problemas
- Consolidar fundamentos para desarrollo frontend
- Preparación para proyectos reales
